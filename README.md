# Bayver Family Recipe Collection

A web-based recipe manager and meal planner with drag & drop functionality.

## Features
- Recipe browser with filters
- Monthly meal planner
- Grocery list generator
- Star ratings and favorites
- Print-friendly views

## How to Use
Open the site in any web browser. No installation needed.

## Adding Recipes
Send recipe links or text to your assistant and they'll be added automatically.
